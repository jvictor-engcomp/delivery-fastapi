    requisição e nomear rota. usar função async e return

4° models
    aqui tratamos do banco de dados. Usar sqlalchemy, importar create_
    engine que serve para linkar com o banco de dados, importar 
    declarative_base que serve para criar a base do banco e o orm, 
    importar tipos como Column, Integer... ForeignKey. Criar classes que
    serão as tabelas, por a base como parâmetro, definir as colunas 

5° models -> alembic
    alambic serve para mexer com migrações/atualizações do banco de dados
    comandos:
        alambic init alambic(alterar link e base posteriormente)
        alambic revision --autogenerate -m ""
        alambic upgrade head

    pode adicionar. Para fazer alguma resquisição no banco de dados, tem 
    que existir uma função que passa uma sessao como dependência no 
    parâmetro. Importar Depends do fastapi.

7° dependencies

    Sessão com sessionmaker, importada do sqlalchemy.orm e passar como
    bind = db. Usar yield para dar um "return" dessa sessão, isso dentro 
    de um try, dentro do finally fechar a sessão.

8° criptografando senhas
